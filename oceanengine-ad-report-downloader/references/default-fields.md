# Default 巨量营销广告报表 Fields

Use these fields for the default advertising report unless the user provides a different field sheet.

## Dimensions

- 时间-天
- 项目名称
- 单元名称
- 单元ID
- 抖音昵称

## Metrics

- 消耗
- 展示数
- 新增A3(问询)数
- 新增A3(问询)成本
- 看后搜次数
- 看后搜率
- 平均千次展现费用(元)
- 点击数
- 点击率%
- 平均点击单价(元)
- 点赞数
- 评论量
- 分享量
- 不感兴趣数
- 搜索组件点击数
- 搜索组件点击率
- 播放量
- 完播率%

## Requested But Not Found Directly

- 完播量

Do not silently replace `完播量` with `99%进度播放数`. Ask or flag the gap.
